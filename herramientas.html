Automatizaciones

No-code / low-code (rápidas para integrar apps)

  • Activepieces

    alternativa open-source “estilo Zapier? se puede self-hostear o usar en la nube.

  • Make (ex Integromat)

    builder visual muy flexible (routers, arrays), con foco en compliance (GDPR, SOC 2 Type II). Buena relación potencia/precio.

  • Pipedream

    híbrido no-code + code; excelente para APIs con pasos JavaScript/Node y webhooks. Bueno para devs que quieren control.

  • Zapier

    enorme ecosistema (8.000+ apps) y módulos como Canvas y Functions para flujos más complejos. Ideal cuando querés velocidad sin code.

Open-source / self-host (control y privacidad)

  • Huginn

    version hackeable de IFTTT/Zapier, en tu propio servidor, basado en agentes que leen la web y actúan.

  • n8n

    flujos drag-and-drop o con código; self-host u hospedado por ellos. Fuerte para equipos técnicos y agentes/LLMs.

  • Node-RED

    low-code para eventos/IoT; gran comunidad de nodes y despliegue sencillo.

iPaaS enterprise (seguridad, gobierno, escala)

  • Tray.io

    low-code para integraciones complejas y agentes; pensado para equipos de ops y TI.

  • Workato

    plataforma de orquestación e integración enterprise con fuerte enfoque AI/agents.

RPA (robots de escritorio/web + procesos)

  • Automation Anywhere

    RPA cloud con bots y capacidades AI para procesos críticos.

  • Blue Prism

    RPA enterprise con fuerte foco en seguridad y gobierno.

  • Microsoft Power Automate

    combina cloud flows con Desktop (RPA) y cientos de conectores (ecosistema M365).

  • UiPath

    suite RPA muy completa (UI+API, process mining, governance).

Otras útiles según caso

  • Bardeen

    automatización desde el navegador (scraping ligero, prospección, workflows GTM).

  • IFTTT

    simples para hogar/ móvil/pequeñas tareas de negocio.

  • Parabola

    automatización de datos para retail/e-commerce (emails, PDFs, hojas de cálculo, ERP/Shopify).
